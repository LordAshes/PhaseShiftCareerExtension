The Careers folder can be extratced into the existing Phase Shift Careers folder unless you game already has an existing
"The Beatles" career. However, this is typically not necessary since the contents is for reference only (and does not
include the accompanying music which makes it a non-playable career). The intention of the included Careers folder is to
show the naming convention of the html content and to demonstrate how the band name can be extracted and the inserted into
the html content.
